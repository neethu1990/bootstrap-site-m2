JAVA

A programming language

Java is a , class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let programmers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need to recompile.

Java was originally developed by James Gosling at Sun Microsystems. It was released in May 1995 as a core component of Sun Microsystems' Java platform. The original and reference implementation Java compilers, virtual machines, and class libraries were originally released by Sun under proprietary licenses.

There are five principle goals in the creation of Java language

  1. It must be simple, object-oriented and familiar.
  2. It must be robust and secure.
  3. It must execute with high performance.
  4. It must be architecture-neutral and portable.
  5. It must be interpreted,threaded,and dynamic

As of September 2021, Java 8,11 and 17 are supported as Long-term Support(LTS) versions.


Oracle released the last zero-cost public update for the legacy version Java 8 LTS in January 2019 for commercial use, although it will otherwise still support Java 8 with public updayes for personal use indefinitely.Other vendors have begun to offer zero-cost builds of OpenJDK 18 and 8,11 and 17 that are still receiving security and other upgrades.


Sun has defined and supports four editiond of Java targeting different application environments and segmented many of its APIs so that they belong to one of the platforms. The platforms are:

  • Java Card for smart-cards.
  • Java Platform,Micro Edition (Java ME)- targeting environmrnts with limited resources.
  • Java Platform,Enterprise Edition(Java EE)- targeting large distributed enterprise or internet environment.
  • Java Platform,Standard Edition(Java SE)- targeting workstation environments.

    • The classes in the Java APIs are organised into separate groups called packages. Each package contains a set of related interfaces, classes, subpackages and exceptions.
